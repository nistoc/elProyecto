QUICK START (3 минуты):

1. Установка:
   pip install -r requirements.txt

2. API ключ:
   # Windows PowerShell
   $env:OPENAI_API_KEY = "sk-..."
   
   # Linux/Mac
   export OPENAI_API_KEY="sk-..."

3. Скопируйте файл:
   cp ../agent01/transcript.md .

4. Запуск:
   python -m cli.main

5. Результат:
   transcript_fixed.md ✓

═══════════════════════════════════════════════════════════════

КОНФИГУРАЦИЯ (config/default.json):

{
  "input_file": "transcript.md",        // Входной файл
  "output_file": "transcript_fixed.md", // Выходной файл
  "model": "gpt-4o-mini",               // Модель (gpt-4o-mini | gpt-4o | gpt-3.5-turbo)
  "batch_size": 10,                     // Строк за раз
  "context_lines": 3                    // Контекст из предыдущего batch
}
