✅ ВЫПОЛНЕНО:

1. Интеграция pyannote.audio для диаризации спикеров
   - Новый модуль: infrastructure/audio/diarizer.py
   - Новые зависимости: pyannote.audio, pydub
   
2. Модифицирован pipeline.py
   - Добавлен режим диаризации (use_diarization: true/false)
   - Новая логика: диаризация → извлечение сегментов → whisper-1
   - Обратная совместимость с v2.x режимом
   
3. Новый формат результатов (при use_diarization: true)
   - JSON с полями: speaker, start, end, text, words, language
   - Автоопределение языка для каждого сегмента
   - Детальные тайминги слов
   
4. Документация упрощена:

   ✓ docs/CHANGELOG_v3.0.md - краткий список изменений
