# Быстрое руководство: Настройка языков

## 🚀 TL;DR (Самое важное)

### Для русского + испанского (люди переключаются между языками):

**В `config/default.json` укажите:**
```json
{
  "language": null,
  "prompt": "Здравствуйте, как дела? Buenos días, ¿cómo estás? Хорошо, спасибо. Muy bien, gracias."
}
```

✅ **Готово!** Whisper автоматически определит оба языка.

---

## Три сценария использования

### 1️⃣ Мультиязычное аудио (русский + испанский)

```json
{
  "file": "interview.m4a",
  "language": null,
  "prompt": "Здравствуйте! Buenos días! Как дела? ¿Cómo estás? Спасибо. Gracias."
}
```

**Когда использовать:**
- Разговор на двух языках
- Урок языка
- Интервью с переводом

**Используйте готовый конфиг:**
```bash
python -m cli.main --config config/multilingual_ru_es.json
```

---

### 2️⃣ Только русский язык

```json
{
  "file": "meeting.m4a",
  "language": "ru",
  "prompt": "Добрый день, коллеги. Обсудим проект. Иванов, Петрова, Сидоров."
}
```

**Когда использовать:**
- Деловые встречи
- Лекции
- Записи на русском

**Используйте готовый конфиг:**
```bash
python -m cli.main --config config/russian_only.json
```

---

### 3️⃣ Только испанский язык

```json
{
  "file": "podcast.m4a",
  "language": "es",
  "prompt": "Buenos días, colegas. Discutamos el proyecto. García, Martínez, López."
}
```

**Когда использовать:**
- Испанские подкасты
- Уроки только на испанском
- Записи испаноговорящих

## 📝 Как настроить prompt

### Хороший prompt содержит:
1. **Естественные фразы** (не отдельные слова!)
2. **Примеры на обоих языках** (если мультиязычное)
3. **Специфичные термины** из вашего аудио
4. **Имена людей** которые будут упоминаться

### ✅ Хорошие примеры:

**Бизнес-встреча (русский + испанский):**
```json
"prompt": "Добрый день, коллеги. Buenos días, colegas. Обсудим контракт и договор. Discutamos el contrato y el acuerdo. Blockchain, криптовалюта, smart contract."
```

**Урок языка:**
```json
"prompt": "Сегодня мы изучаем испанский. Hoy estudiamos español. Как сказать 'спасибо'? ¿Cómo se dice 'спасибо'? Gracias! Правильно! ¡Correcto!"
```

**С именами:**
```json
"prompt": "Иванов Сергей, García María, Петрова Елена, Martínez José работают над проектом. Trabajan en el proyecto."
```

### ❌ Плохие примеры:

```json
// Только слова (не фразы)
"prompt": "hello, привет, hola, здравствуйте"

// Слишком длинно (>200 слов)
"prompt": "Очень длинный текст на 500 слов..."

// Не тот язык
"prompt": "English only" // для русско-испанского аудио
```

## 🎯 Коды языков

| Язык | Код | В конфиге |
|------|-----|-----------|
| **Русский** | `ru` | `"language": "ru"` |
| **Испанский** | `es` | `"language": "es"` |
| **Английский** | `en` | `"language": "en"` |
| **Автоопределение** | - | `"language": null` |

**Полный список:** [ISO 639-1](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes)

## 🔧 Пошаговая настройка

### Шаг 1: Скопируйте конфиг

```bash
# Для мультиязычного
cp config/multilingual_ru_es.json config/my_config.json

# Или для русского
cp config/russian_only.json config/my_config.json
```

### Шаг 2: Отредактируйте файл

```json
{
  "file": "ваш_файл.m4a",  // ← Укажите ваш файл
  "language": null,         // ← null для автоопределения
  "prompt": "..."           // ← Ваш prompt
}
```

### Шаг 3: Запустите

```bash
cd agent03
python -m cli.main --config config/my_config.json
```

### Шаг 4: Найдите результаты

```
processing_workspaces/
└── ваш_файл/
    └── output/
        ├── ваш_файл_transcript.json  ← Здесь результат!
        └── ваш_файл_transcript.md
```

## ❓ Частые вопросы

### Q: Можно ли указать больше двух языков?
**A:** Да! Используйте `language: null` и добавьте примеры всех языков в prompt:
```json
"prompt": "Hello! Привет! ¡Hola! Bonjour!"
```

### Q: Нужно ли всегда указывать prompt?
**A:** Нет, но **очень рекомендуется** для:
- Мультиязычного аудио
- Специфичной терминологии
- Правильного написания имен

### Q: Что лучше: `language: "ru"` или `null`?
**A:**
- `"ru"` - **точнее** для моноязычного русского
- `null` - **гибче** для мультиязычного или неизвестного языка

### Q: Как проверить, какой язык распознан?
**A:** Смотрите в JSON-результате:
```json
{
  "text": "Hola, buenos días",
  "language": "es"  ← Распознанный язык
}
```
